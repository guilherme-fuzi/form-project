Neste projeto foi utilizado o Lombok, portanto é necessária a instalação

https://projectlombok.org/download

Criar um banco de dados com o nome "db_formulario_qintess"

Além disso o modelo lógico do banco está na pasta Docs, junto com as queryes do banco para que a aplicação funcione.

A modelagem/análise do banco para o desenvolvimento do projeto foi baseada no arquivo "Piloto_PS_V2.xlsx".
